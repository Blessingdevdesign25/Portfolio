# Personal Portfolio

Welcome to my personal portfolio! This project showcases my design and development capabilities. It was initially designed in Figma and beautifully translated into a high-performance React web application.

## 🚀 Tech Stack

The framework consists of modern standard tooling for optimized front-ends:
- **React 18** – Core component framework.
- **Vite** – Next-generation build tool configured for extremely fast Hot Module Replacement (HMR).
- **Tailwind CSS & Material UI** – For highly customizable, responsive, and robust semantic styling.
- **Radix UI** – Unstyled headless components ensuring maximum accessibility.
- **Framer Motion** – Spring-based animations for delightful interactions.

## 📦 Project Structure

```text
Portfolio/

│   ├── assets/        # Media, fonts, and SVG files

```

## 🛠️ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine. We recommend managing your packages through npm.

### Installation

To install all dependencies locally, navigate to the repository directory and run:

```bash
npm install
```

### Running the Dev Server

To preview the portfolio locally, start the Vite development environment:

```bash
npm run dev
```

The application should automatically bind to the nearest accessible local port, primarily: `http://localhost:5173`. 

## ✨ Key Features

- **Responsive Grid Design**: A mobile-first approach ensuring it looks stunning across all hardware profiles.
- **Framer Animations**: Scroll and click event micro-interactions powered by Framer.
- **Accessibility**: ARIA-labeled components and structured navigation per Radix standards.
